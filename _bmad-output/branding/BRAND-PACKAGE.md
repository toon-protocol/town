## Origin Story

> TOON Protocol — the Token-Oriented Open Network. A protocol stack where value moves like data and content flows like water. ILP carries payments the way TCP carries packets. TOON delivers content the way HTTP delivers pages. Three kinds of tokens converge in one network: crypto tokens pay the toll, inference tokens carry the thought, data tokens hold the structure. Where every token counts.

## Triple-Token Thesis

| Token type | Role in stack |
|-----------|--------------|
| **Crypto tokens** | Payment — value transfer via ILP channels |
| **Inference tokens** | Intelligence — AI agents as first-class publishers |
| **Data tokens** | Structure — TOON-encoded content units |

### Mark Concept

- Four squares = four towns = four nodes in the network
- Each town has unique internal structure = each node has its own data
- White squares = lit windows = activity / data flowing
- Gold square = the token = value in the system
- Cream channels between towns = network routes (ILP)
- Modular: crop to 1 tile (favicon), 2 tiles (peering), 4 tiles (full network)

### Visual Continuity with TOON Format

- Inherits palette from [toonformat.dev](https://toonformat.dev): `#fef3c0` / `#1b1b1f` / `#fff`
- TOON format mark = one block of structured data
- TOON Protocol mark = four blocks of structured data, connected
- Format is a document. Protocol is a network of documents.
- Same visual language, different scale.
