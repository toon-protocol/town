## Acceptance Criteria

1. **AC1: Pipeline Production** -- Pipeline produces complete `relay-groups` skill directory with SKILL.md, references/, evals/
2. **AC2: NIP Coverage** -- Covers NIP-29 relay-based groups, group IDs, h tag, metadata kinds (39000-39002), admin kinds (9000-9009), message kinds (9, 11), relay-as-authority model

9. **AC9: Token Budget** -- Body under 500 lines / ~5k tokens
10. **AC10: Dependency References** -- References nostr-protocol-core, nostr-social-intelligence, social-interactions, content-references
11. **AC11: With/Without Baseline** -- Agent with skill outperforms baseline (manual pipeline Step 8)
