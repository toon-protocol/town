# @crosstown/core

Core library for Nostr-based ILP peer discovery, bootstrap, and settlement negotiation.

## Install

```bash
npm install @crosstown/core
```

## Usage

```ts
import {
  NostrPeerDiscovery,
  BootstrapService,
  SocialPeerDiscovery,
  parseIlpPeerInfo,
  buildIlpPeerInfoEvent,
} from '@crosstown/core';
```
