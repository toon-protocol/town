# @toon-protocol/core

Core library for Nostr-based ILP peer discovery, bootstrap, and settlement negotiation.

## Install

```bash
npm install @toon-protocol/core
```

## Usage

```ts
import {
  NostrPeerDiscovery,
  BootstrapService,
  SocialPeerDiscovery,
  parseIlpPeerInfo,
  buildIlpPeerInfoEvent,
} from '@toon-protocol/core';
```
